**Small category / status pill.** Use for赛道 (category) labels, plan badges, "推荐" flags.

```jsx
<Badge tone="coral">旅游攻略</Badge>
<Badge tone="sun" variant="solid">推荐</Badge>
```

Tones: `coral` / `sprout` / `sun` / `grape` / `neutral`. `variant="soft"` (tinted, default) for quiet labels; `variant="solid"` (filled gradient) to draw the eye.
