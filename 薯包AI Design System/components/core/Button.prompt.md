**Chunky 3D jelly button — use for any primary action or CTA.** Coral gradient with a depressible bottom edge and springy press.

```jsx
<Button variant="primary" size="lg" icon={<Sparkles size={18} />}>一键生成</Button>
```

Variants: `primary` (coral gradient, default), `sun` (sunset gradient), `sprout` (green), `secondary` (white with coral outline), `ghost` (text only). Sizes: `sm` / `md` / `lg`. Use `block` for full-width, `iconRight` for trailing icons. Reserve `primary`/`sun` for the single most important action per view.
